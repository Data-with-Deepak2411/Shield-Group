# Shield-Group
🎤 Presentation Script: ShieldGroup Dashboard (PwC Virtual Case Experience)

Introduction

Hello, my name is Deepak Kumar, and I’m a Data Analyst with hands-on experience in Power BI, SQL, and Advanced Excel.
Today, I’ll be presenting my analysis of the ShieldGroup insurance dataset, developed as part of the PwC Virtual Case Experience on Forage.

Project Objective

The main objective of this project was to analyze insurance claims data and provide insights into company performance, claim trends, and payment patterns.
This dashboard helps stakeholders make data-driven decisions by identifying key metrics and trends.

Dashboard Overview

This interactive Power BI dashboard is divided into multiple sections:

Filters for Company, Segment, and Underwriter
KPI cards for overall performance
Visualizations for claims, payments, and geographic distribution

Key Metrics (KPIs)

On the right-hand side, we can see three key performance indicators:

Total Payment Amount: 23.44 Million
Number of Claims: 4,923
Number of Policies: 34

These KPIs give a quick snapshot of the business performance.

Claims & Payment Analysis

In the central table, we analyze:

Number of claims per company
Average payment amount

This helps identify which companies have higher claim frequency and cost impact.

Underwriter Performance

The bar chart shows the amount handled by each underwriter per year.
This allows us to compare performance across years and identify top contributors.

Geographical Insights

The map visualization displays claim amounts by city.
We can observe that most claims are concentrated in specific regions, helping in regional risk assessment and resource allocation.

Trend Analysis

The line chart shows:

Count of claims over time
Total payment amount trend

We can clearly see an increasing trend in both claims and payments, indicating business growth but also rising risk exposure.

Key Insights

From this dashboard, we can derive several insights:

Certain companies and regions contribute more to claims
Payment amounts are increasing over time
Some underwriters consistently handle higher volumes

Conclusion

In conclusion, this dashboard provides a comprehensive view of insurance operations, helping stakeholders monitor performance, identify risks, and make strategic decisions.
